4. Flujo especial para tres zonas con Unity
-------------------------------------------
Se preparo un comportamiento especial SOLO para:
- Biblioteca
- Laboratorios
- Plazoleta Central / Cafeteria

Flujo implementado:
1. Usuario presiona el pin.
2. Se abre el panel de informacion de la zona.
3. Antes de las actividades aparece un bloque "Reto Unity".
4. El boton "Iniciar reto" abre Unity en un modal encima del mapa.
5. Las actividades de descubrimiento estan bloqueadas mientras no se complete
   el reto.
6. Cuando Unity emite GameOver, la zona se desbloquea.
7. Al desbloquearse, las actividades quedan disponibles.

Boton temporal:
- Se agrego un boton "Completar prueba" dentro del modal Unity.
- Sirve para validar el desbloqueo mientras el build Unity final no este listo.
- Este boton debe eliminarse o esconderse cuando Unity ya dispare GameOver
  correctamente desde sus minijuegos reales.

Archivo principal modificado:
- src/components/ExpedicionUAOview.tsx
